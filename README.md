# Slutprojekt JavaScript

	Titel: Magic Card Finder


## Description

	Sidan är till för att kunna söka och hitta Magic kort 	med hjälp av Magic: The Gathering's Scryfall-API.
	Det finns också information där, genom att klicka på en 	länk, om hur man kan bygga egna lekar på bästa sätt.
	Sidan är inte responsiv och är gjord för att enbart 	kunna visas på dator-skärmar.

## Features
	
* Visar en lista av alla korten på vänstra sidan av webbplatsen.

* Visar ett sökningsbart fält där användaren anger en input som består av text. När Enter-knappen trycks, ska listan på alla korten som matchar visas som en lista, där de även är klick-bara för att få fram bilden av det valda kortet.

* Den första länken i navigeringsbaren ska öppna upp en annan sida som har information om hjälpsamma sätt att bygga egna lekar.
